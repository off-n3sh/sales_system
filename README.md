# Order Management System

A lightweight, open-source order management system for small to medium businesses.

## Tech Stack

**Frontend**
- HTML
- JavaScript
- Jinja2 templating

**Backend**
- Python
- Flask (with Jinja2)

**Database**
- MongoDB
